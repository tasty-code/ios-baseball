**Learn and proceed**

컬렉션 타입, 함수 네이밍, 난수 생성 등을 학습하며 진행했다.

컴퓨터에서 지정해주는 임의의 수와 9개의 사용자의 임의의 수를 비교하는 숫자 야구게임 프로젝트를 진행했다.
